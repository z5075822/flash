# Flash
This app was designed to assist students in their learning and understanding of INFS3634 Business Process Management. It includes two self-test features (1) a multiple choice quiz and (2) flashcards to assist with the learning of content.

# Build instructions

1) download zip from github
2) unzip file
3) open android studio 
4) file - open > navigate and locate 'flash-master' file
5) open in new window
